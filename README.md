# 🌲 Timber Game

A simple and fun **Timber-style arcade game** where the player chops a tree while avoiding branches. Built as a practice project using c++ and sfml to improve programming logic and game development skills.

---

## 🎮 Gameplay

* Control the player to chop the tree from left or right
* Avoid incoming branches
* Survive as long as possible to increase your score
* Game ends when you hit a branch or timer runs out ⛔

---

## 🛠️ Tech Stack

* Language: C++
* Library/Framework: SFML 

---

## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/AbhijitCoder/timber-game.git
   ```

2. Navigate to the project folder:

   ```bash
   cd timber-game
   ```

3. Run the game:

   ```bash
   g++ Game.cpp -o Game -sfml-system -sfml-window -sfml-graphics -sfml-audio
   ```
